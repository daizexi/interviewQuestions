   - 深色模式如何实现的

     - 主要使用了scss的mixin、插值语句、属性选择器将色值注册在body元素，再使用var引用这些色值

       - mixin是用于将色值注入body元素

         ```{scss}
         @mixin theme($name='')
         {
           @if name == '' {
             body{
               @content;
             }
           }
           @else{
             body[data-theme=`#{$name}`]
           }
         }
         ```

         属性选择器将区分深色模式和普通模式，一旦启用深色模式，只需要使用document.body.setAttribute()为body元素添加一个属性即可。
